## Como testar esse projeto?

1. **Clonar o Repositório**
```bash
git clone https://github.com/seu-usuario/upload-backend
cd api-mongo
```
2.**Instalar as dependências**
```bash
npm install
```
3.**Executar**
```bash
npm run dev
http://localhost:3000
```
